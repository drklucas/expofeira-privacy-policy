APP - EXPOFEIRA

TERMOS E CONDIÇÕES GERAIS DE USO

DO APLICATIVO "EXPOFEIRA"

Este Termo de Uso estabelece as condições para o uso do aplicativo Expofeira relacionados aos dados pessoais coletados do Usuário. Ao utilizar o aplicativo Expofeira você concorda e aceita todas as disposições contidas neste Termo.

1. Coleta de dados pessoais

O aplicativo Expofeira coleta os seguintes dados pessoais do usuário: nome, CPF, data de nascimento, telefone e e-mail. Essas informações são necessárias para realizar o cadastro no aplicativo e proporcionar uma experiência personalizada ao usuário.

2. Medidas de segurança

O aplicativo Expofeira implementa medidas de segurança adequadas para proteger os dados pessoais do usuário contra acesso não autorizado uso indevido ou divulgação. No entanto, é importante ressaltar que nenhum sistema ou método de transmissão de dados pela internet é totalmente seguro e não podemos garantir a segurança absoluta das informações transmitidas por meio do aplicativo.

3. Armazenamento dos dados pessoais

Os dados pessoais coletados serão armazenados pelo aplicativo Expofeira pelo tempo necessário para atingir os propósitos mencionados no Aviso de Privacidade, disponível neste Aplicativo Expofeira.

4. Direitos do usuário

O usuário tem o direito de acessar corrigir atualizar e excluir seus dados pessoais a qualquer momento mediante solicitação por escrito enviada para o endereço de contato indicado no aplicativo. O usuário também pode optar por cancelar o recebimento de comunicações de marketing do aplicativo Expofeira, caso esteja disponível.

5. Alterações no Termo de Uso

Reservamo-nos o direito de alterar este Termo de Uso a qualquer momento visando refletir mudanças legais e/ou regulatórias ou em função de atualizações no aplicativo. Recomendamos ao usuário que revise regularmente este Termo, para manter-se atualizado sobre suas disposições.

6. Contato

Em caso de dúvidas ou preocupações relacionadas à privacidade e proteção de dados pessoais entre em contato conosco por meio das informações fornecidas dentro do aplicativo Expofeira.

Ao utilizar o aplicativo Expofeira você concorda e aceita todas as disposições deste Termo de Uso.

Data de vigência: 31 de dezembro de 2025
